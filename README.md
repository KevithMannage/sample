GuidelineX 

1. Open VS Code.
2. Open a new terminal.
3. Run the following command in the terminal to clone the repository
   ```bash
   git clone https://github.com/KevithMannage/Devthon.git
4. To start the frontend 
    ```bash
    cd frontend
    ```
    ```bash
    npm install
    ```
    ```bash
    npm run dev
    ```
5. To start the backend
   (open new terminal)
    ```bash
    cd backend
    ```
    ```bash
    npm install
    ```
    ```bash
    npm run start
    ```
